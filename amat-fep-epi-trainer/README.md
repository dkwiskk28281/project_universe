# FEP/EPI Customer Engineer Trainer

Applied Materials FEP/EPI(RTP/Epitaxy) Customer Engineer 입사 전 예습을 위한 정적 웹앱입니다.

## 실행

`index.html`을 브라우저로 열면 됩니다.

## 포함 기능

- 4주 학습 로드맵과 완료 체크
- RTP/Epitaxy/서브시스템/설치 플로우 학습 맵
- 현장 고장 대응 시나리오
- 퀴즈와 핵심 용어장
- 브라우저 `localStorage` 기반 진행률 저장

## 학습 범위

공개 채용공고와 Applied Materials 공개 제품/기술 소개 페이지를 기준으로 구성했습니다. 실제 장비별 세부 절차, 내부 매뉴얼, 고객사 site rule은 입사 후 공식 교육과 현장 지시를 우선해야 합니다.
