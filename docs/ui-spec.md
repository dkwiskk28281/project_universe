# UI Spec

## Product Feel

Professional, calm, technical, and motivating. The UI should reduce math anxiety without becoming childish. Semiconductor field language is allowed, but first use of hard terms should include easy explanations.

## MVP Navigation

- Dashboard: readiness, today's learning, course spine.
- Diagnostic: adaptive ratio/unit questions.
- Lesson: concept blocks in `현상 -> 그림 -> 숫자 -> EPI 연결` order.
- Practice: gas-flow ratio slider and immediate feedback exercise.
- Review: generated spaced repetition cards.
- AI Tutor: structured explanation and local packet.
- Knowledge Map: prerequisite graph cards.

## Accessibility

- Dark theme with high contrast.
- Keyboard-clickable buttons.
- Text labels are present; color alone does not carry meaning.
- Cards wrap on mobile.
- No small fixed-width tables in the MVP.

## Responsive Rule

At widths below 860 px, all major grids collapse to one column. Flow strips also collapse so labels do not overlap.
