# Knowledge Graph

## Node Contract

Each `KnowledgeNode` has:

- `id`
- `slug`
- `titleKo`
- `titleEn`
- `domain`
- `level`
- `prerequisites`
- `learningObjectives`
- `estimatedMinutes`
- `masteryThreshold`
- `theoryWeight`
- `practiceWeight`
- `applicationWeight`
- `tags`

## Unlock Rule

The system can preview locked nodes, but mastery progress should focus on unresolved prerequisites until the required threshold is reached.
